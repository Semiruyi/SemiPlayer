# Test Media Fixture

`sample.mp4` is a 3-second MP4 fixture for FFmpeg integration tests and the C ABI smoke host.

- Video: H.264, 320x180, 15 fps
- Audio: AAC, 48 kHz, mono

The integration test verifies that both audio and video streams are detected. Confirm that this
derived media may be redistributed before publishing the repository or any release artifact that
includes it.
